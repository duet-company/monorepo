# Duet Company - Frontend

React + TypeScript web dashboard for Duet Company platform.

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm run start
```

## 📁 Structure

```
src/
├── components/    # Reusable React components
├── pages/        # Next.js pages
└── services/     # API client services
```

## 🛠️ Tech Stack

- **Framework:** Next.js 14
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Data Fetching:** TanStack Query
- **HTTP Client:** Axios
